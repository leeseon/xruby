XRuby
Xue Yong Zhi (zhixueyong AT hotmail.com)
http://www.xruby.com

Overview
=====================
Compile Ruby to Java Bytecode


Usage
=====================
Compile then run:
>java -jar xruby-0.1.0.jar -c test.rb
>java -jar test.jar

Run script directly (compile behind the scene):
>java -jar xruby-0.1.0.jar test.rb
